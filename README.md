# 🏢 Employee Management System (EMS)

A full-stack Employee Management System built using **Spring Boot**, **Thymeleaf**, **TailwindCSS**, and **MySQL**.

The application helps organizations manage employee records efficiently with features like employee registration, updates, department management, and responsive dashboards.

---

## 🚀 Features

* ✅ Add New Employees
* ✅ Update Employee Details
* ✅ Delete Employees
* ✅ Employee Search Functionality
* ✅ Department Management
* ✅ Responsive Admin Dashboard
* ✅ Clean UI with TailwindCSS
* ✅ Server-side Rendering using Thymeleaf
* ✅ MySQL Database Integration
